About This Application

This is a simple yet powerful Inventory Management System designed to help users efficiently manage their products. It provides a user-friendly interface to view, add, edit, and delete products with ease.

Technology Stack

  • Front-end: Developed using Angular for dynamic UI updates.
  • Back-end: Uses JSON Server for handling API requests.
  • Authentication: Implemented using JWT (JSON Web Token) for secure login.
  • UI Framework: Styled with CSS & Bootstrap for a modern, responsive design.

How It Works

The system allows authenticated users to manage products efficiently. Upon logging in, users can view a list of products, search for specific items, and perform CRUD (Create, Read, Update, Delete) operations.

Future Enhancements: Plans include integrating a database like MySQL, adding role-based access, and implementing RESTful APIs for improved scalability.
